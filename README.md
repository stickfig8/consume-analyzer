# 소비 구조 분석 리포트 📊

## 🖥️ 프로젝트 소개

LLM 기반 분석을 통한 감정 소비 리포트 서비스입니다.

## 📅 개발기간

26.02. ~ 26.03.

## 👥 개발 인원

개인 프로젝트

## ⚙️ 주요 기능

- 소비 입력 기반 분석 요청
- 누락 입력 validation
- 더미데이터 테스트 기능
- 서버리스 함수를 통한 LLM 요청
- LLM 요청 단계별 로딩 창 & 요청 중단
- LLM 기반 소비 구조 분석 리포트 제공
  - 감정 소비 점수 및 각 소비 별 비교 그래프
  - 현재 소비 구조 및 비율에 대한 인사이트
  - 현재 소비 현황의 개선점 추천
  - 소비 구조의 리스크 예측
  - 중요 키워드 카테고리 별 별도 디자인
- 소비 리포트 pdf export
- 반응형 디자인 적용
