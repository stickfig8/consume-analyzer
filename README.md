# Consume-Anaylzer 📊

## 🖥️ 프로젝트 소개

LLM 기반 분석을 통한 감정 소비 리포트 서비스입니다.

## 📅 개발기간

26.02. ~ 26.03.

## 👥 개발 인원

개인 프로젝트

## ⚙️ 주요 기능

- 소비 입력 기반 분석 요청
- 누락 입력 validation
- 서버리스 함수를 통한 LLM 요청
- 더미데이터 테스트 기능
- 소비 리포트 열람
- 소비 리포트 pdf export
- LLM 요청 단계별 로딩 창 & 요청 중단
- 반응형 디자인 적용
